# מערכת הצעות מחיר · bidernet · v2.1

מערכת עצמאית ל-quotation.bidernet.co.il.

## פיצ'רים
- דשבורד, סטטוסים חכמים (נשלח/נצפה/חתום/פג תוקף), שכפול, מחיקה, הערות פנימיות
- שליחה בוואטסאפ + תבנית הודעה עם {שם} ו-{קישור} + כפתור "העתק קישור"
- התראות וואטסאפ (צפייה/חתימה), מספור נפרד, חיפוש
- אייקונים וקטוריים נקיים, סרגל צד עם לוגו ופונט PingHL
- גרסת מובייל כאפליקציה (PWA) + סרגל ניווט תחתון + פאביקון בכל הדפדפנים
- חתימה דיגיטלית, הפקת PDF (הדפסה מהדפדפן)

## התקנה
1. cPanel → Subdomains → quotation ; MySQL → מסד+משתמש חדשים
2. העלה את כל הקבצים ל-quotation.bidernet.co.il/
3. cp config.sample.php config.php ומלא DB + APP_URL
4. גלוש ל-install.php → מחק אותו
5. הגדרות → GREEN API + טלפון להתראות + תבנית וואטסאפ
