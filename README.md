# מערכת הצעות מחיר · bidernet (עצמאית) — v1.1

מערכת עצמאית ל-quotation.bidernet.co.il — נפרדת מה-CRM, עם מסד נתונים משלה.

## התקנה
1. cPanel → MySQL Databases → צור מסד + משתמש חדשים (למשל biderne1_quotes).
2. העלה את כל הקבצים לתיקיית הסאב-דומיין (כולל התיקייה lib/tcpdf).
3. העתק config.sample.php ל-config.php ומלא DB + APP_URL.
4. גלוש ל-install.php (יוצר/מעדכן טבלאות) → מחק אותו אחרי.
5. הגדרות → הזן idInstance + apiTokenInstance (GREEN API) + מספר בעל העסק להתראות.

> שדרוג מ-1.0: פשוט העלה את הקבצים החדשים וגלוש ל-install.php פעם אחת.
> המיגרציה מוסיפה עמודות חדשות בלבד (sent_at, viewed_at, internal_notes, notify_*)
> ולא מוחקת נתונים קיימים.

## מה חדש ב-1.1
- **עיצוב חדש**: סרגל צד, לוגו, פונט PingHL, צבעי מותג (#deedce / #464646).
- **לוח בקרה (דשבורד)**: שווי הזמנות חתומות, אחוז סגירה, ומסמכים שדורשים טיפול.
- **הזמנות חתומות**: קטגוריה נפרדת בסרגל הצד.
- **סטטוסים חכמים**: טיוטה · נשלח · נצפה · חתום · פג תוקף (מחושב אוטומטית).
- **תוקף אוטומטי**: לפי ימי עסקים (מדלג על שישי/שבת).
- **שכפול הצעה**: כפתור ⧉ שמעתיק הצעה כטיוטה חדשה.
- **הערות פנימיות**: לכל הצעה, נראות רק בניהול (לא ללקוח).
- **התראות וואטסאפ**: לבעל העסק כשהלקוח צופה בהצעה וכשהוא חותם.
- **הפקת PDF אמיתית**: קובץ PDF מלא בעברית (TCPDF) לצפייה, הורדה ושליחה בוואטסאפ.

## גישה
- ACCESS_CODE ריק ב-config = בלי מסך כניסה. מלא קוד = כניסה עם קוד (עם כפתור יציאה).
- sign.php (חתימת לקוח) תמיד ציבורי.

## הערות טכניות
- הפקת PDF מבוססת על ספריית TCPDF המצורפת ב-lib/tcpdf (לא צריך composer).
  דורשת את התוספים הסטנדרטיים של PHP: mbstring + gd (קיימים כברירת מחדל ב-cPanel).
- אם ה-PDF לא נוצר: ודא שתיקיית ה-tmp של השרת ניתנת לכתיבה
  (TCPDF משתמש ב-upload_tmp_dir או sys_get_temp_dir לזיכרון מטמון).
- התיקייה lib/ חסומה לגישה ישירה מהדפדפן (lib/.htaccess).

## קבצים עיקריים
- index.php — לוח בקרה · quotes.php — רשימת הצעות/הזמנות
- quote_form.php — יצירה/עריכה · quote_view.php — תצוגה · pdf.php — הפקת PDF
- sign.php — חתימת לקוח (ציבורי) · send.php — שליחה בוואטסאפ · quote_dup.php — שכפול
- catalog.php — קטלוג שירותים · settings.php — הגדרות · lib/ — ליבה + TCPDF
